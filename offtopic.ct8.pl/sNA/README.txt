Requirements
------------------------------
PHP >= 5.4
MCrypt Extension
MBString Extension
FileInfo Extension
GD Extension

Installation
------------------------------

(This copy of SANS has been optimised for cPanel servers, if you'd prefer the raw version, please contact support!)

1. Upload contents of ZIP file to the root directory of your webhost
2. Navigate to your website in your web browser and follow the installation wizard steps
